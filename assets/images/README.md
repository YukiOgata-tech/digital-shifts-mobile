# Dミセ モバイルアプリの画像資産

このディレクトリには、ネイティブアプリから直接参照する画像を配置します。

## 配置ルール

- `brand/`: Dミセ共通のロゴ・ブランド素材
- `icons/`: iOS / Android のアプリアイコンと通知アイコン
- `illustrations/`: スタッフ向け画面で用いる説明用イラスト

## 現在の資産

| ファイル | Web版の元ファイル | 用途 |
| --- | --- | --- |
| `brand/dmise-logo.png` | `public/images/dmise-logo-trans.png` | ログイン画面、ヘッダーなどアプリ内のブランド表示 |

`brand/dmise-logo.png` はWeb版と共通の元データです。ストア用アプリアイコンには、余白・背景色・Androidのアダプティブアイコン用レイヤーを別途用意してから使用します。
